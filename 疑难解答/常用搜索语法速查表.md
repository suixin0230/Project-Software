## 常用高级搜索语法速查表

| 语法        | 说明         | 示例                   |
| ----------- | ------------ | ---------------------- |
| `site:`     | 限定域名     | `site:edu.cn 人工智能` |
| `filetype:` | 限定文件类型 | `报告 filetype:docx`   |
| `intitle:`  | 标题含关键词 | `intitle:年终总结`     |
| `inurl:`    | URL含关键词  | `inurl:admin`          |
| `intext:`   | 正文含关键词 | `intext:用户手册`      |
| `-`         | 排除词       | `手机 -苹果`           |
| `""`        | 精确匹配     | `"大数据应用"`         |
| `*`         | 通配符       | `人工智能 * 技术`      |
| `OR`        | 或运算       | `AI OR 人工智能`       |
| `cache:`    | 网页缓存     | `cache:example.com`    |
| `related:`  | 相似网站     | `related:zhihu.com`    |
